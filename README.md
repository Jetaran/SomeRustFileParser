## SomeRustFileParser

Библиотека для парсинга специально размеченных файлов, содержащих список банковских транзакций.

### Поддерживаемые форматы

*.txt
*.csv
*.bin

### Основной интерфейс

Структура Parser, возвращающая вектор с записями транзакций.

Запись транзакции представлены в виде структуры TransactionRecord.

### Использование

```rust
let parser = Parser::new()
let transactions = parser.parse_file(file_path)
```
### Утилиты

#### 1. Comparer

Сравнивает парами файлы поддерживаемых форматов и выводит различия в формате списка уникальных идентификаторов транзакций.
Принимает чётное количество аргументов.

##### Запуск:

```bash
comparer --file1 file1_path --file2 <file2_path> ... --fileN <fileN_path> --fileN+1 <fileN+1_path>
```
